Carregamento de Dados FTTH

Como resolver problemas de carregamento de dados

Esta página ajuda a diagnosticar e resolver problemas com o carregamento de dados do banco de dados para o sistema FTTH.

O que fazer se seus dados não estão aparecendo:

  1. Use esta página para verificar a autenticação e tentar carregar os dados diretamente do banco
  2. Se esta página conseguir carregar os dados, clique em "Continuar para Página Principal"
  3. Se ainda tiver problemas, você pode usar os seguintes arquivos auxiliares:
    • auth-fix.js - Corrige problemas com autenticação
    • fix-load-data.js - Força o carregamento de dados do banco
    • Pagina1-com-fix.html - Versão da página principal com todas as correções aplicadas
