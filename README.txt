INSTRUÇÕES DE USO DA PLANILHA:

1. Preencha os dados na planilha seguindo o formato estabelecido.
2. Campos obrigatórios: Cidade, Tecnico (para técnicos sem veículo) ou Placa (para registros de veículos).
3. Na operação, utilize apenas 'BJ Fibra' ou 'Megalink'.
4. Salve a planilha no formato CSV (separado por ponto e vírgula) para importação no sistema.
5. Se registrar um técnico sem veículo, deixe o campo Placa e Modelo vazios.
6. O formato de data para início de contrato deve ser AAAA-MM-DD (ex: 2023-05-01).
7. Coordenadas de latitude e longitude devem usar ponto como separador decimal.

Seus dados serão carregados automaticamente no sistema quando importados.
